Il progetto

Toolkit ComfyUI di 11 custom nodes Python per la generazione e la pipeline di character AI. Costruito durante lo sviluppo di Marty (character training Z-Image LoRA) e poi rilasciato come open source MIT per la community ComfyUI italiana. v1.0.0 lockata 29 apr 2026 con refactor monolite 3592 righe → 9 moduli + shim re-export per backward compat.

11 nodes registered

  • CharacterPromptBuilder — builder modulare prompt character con 30+ pool tematici (face shape, hair, eyes, expression, clothing, lighting, mood)
  • CharacterPromptBuilderFromMetadata — rebuild prompt da metadata sidecar JSON per reproducibility
  • CharacterSaveTextFile + CharacterLoadMetadata — persistenza JSON sidecar per workflow chain
  • PortraitFaceBuilder — face-only mode con 10 lighting patterns canonici fotografia (butterfly, Rembrandt, split, loop, broad, short, clamshell, Paramount)
  • SceneContextBuilder — context generator scene con body_type lock, location, mood, time-of-day
  • LookVariationBuilder — 7 attribute pools per varianti hair/makeup/accessories face-locked
  • WhatsChangeBuilder — modular attribute change con NONE skip pattern + anchor instruction anti-drift
  • LogoPromptBuilder + LogoQualityGate + LogoBatchSaver — logo pipeline 3-node con quality gate automatico

Casi d'uso

Pipeline character generation Marty (LoRA training Z-Image De-Turbo, ai-toolkit), product character viz commerciali, ricerca creative pre-shoot, generazione dataset training per AI custom client (architetti virtual, e-commerce shapewear).
