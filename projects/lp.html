Il progetto

Brand commercial LP Group con character protagonista: un camaleonte 3D fotorealistico animato. Lavoro di rigging end-to-end del creature + character animation sullo spot: body mechanics, sistema facciale, tongue dynamics (lingua proiettile), spline coda e performance animata frame-by-frame. Pipeline integrata SETPOINTSTUDIO, controlli rig intuitivi e deformazioni stabili sotto pose estreme.

Il mio contributo — rigging + animation

  • Body mech rig — skeleton quadruped con IK/FK switch + stretchy limbs · controllo digit-by-digit dita prensili (le 5 dita del camaleonte hanno opposable grip caratteristico)
  • Tongue dynamics — sistema spline a curvatura variabile per la lingua-proiettile, con squash & stretch automatic + curl/uncurl progressivo via slider
  • Tail spline — coda prensile con FK chain + dynamic overlap secondary, capace di curl-roll su sé stessa
  • Facial rig — eyes independent (camaleonte ha occhi che ruotano indipendenti 360°), eyelids blendshapes, jaw open + tongue protrude attached
  • Skin shading prep — UV layout + topology ottimizzata per displacement scales + chromatophores layer (cambio colore via texture animation)
  • Python tools — auto-rig snippets per accelerare iterazioni revisioni
  • Character animation — performance animata dello spot: body acting, facial expressions, tongue strike, tail balance secondary motion. Animazione keyframe + cleanup graph editor

Sfide tecniche

Il camaleonte ha 3 caratteristiche anatomiche fuori standard quadruped che richiedono custom rigging: occhi independenti (non possono usare aim-constraint condiviso), lingua ballistic projectile (impossibile da animare keyframe puro, deve essere procedural), e dita zygodactyl (2 davanti + 3 dietro su mano, opposable). Ognuna ha richiesto un sub-system dedicato.

Tools

Maya · Python · MEL · custom auto-rig toolset · ZBrush (sculpt review) · Substance Painter (shading prep)
